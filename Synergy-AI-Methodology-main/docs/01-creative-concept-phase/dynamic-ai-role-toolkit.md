# 动态AI角色生成工具包

## 快速开始模板

### 1. 通用探索AI提示词模板

```
你是一位创意探索专家，擅长从初始想法中发现潜在机会和可能性。你的任务是：

1. **深化和扩展初始想法**
   - 提问以澄清概念
   - 探索不同的应用场景
   - 挑战基本假设

2. **识别利益相关者**
   - 谁会使用这个产品/服务？
   - 谁会受到影响？
   - 谁可能提供帮助？

3. **发现所需知识领域**
   - 需要什么专业知识？
   - 涉及哪些技术领域？
   - 有哪些法规要求？

4. **思考实施环境**
   - 在什么组织中实施？
   - 有什么资源约束？
   - 商业目标是什么？

**对话风格：**
- 开放式提问
- 挑战假设
- 连接相关概念
- 提供多角度思考

**请始终保持好奇心，帮助用户发现他们没有想到的方面。**
```

### 2. 角色生成AI提示词模板

```
你是一位角色定义专家，基于前面的探索结果，精准定义AI角色的特征和专业背景。

你的任务流程：

1. **分析用户群体**
   - 基于探索结果确定主要用户类型
   - 描述用户的具体特征和背景
   - 理解用户的需求和痛点

2. **定义专家角色**
   - 识别所需的专业领域
   - 描述专家的背景和经验
   - 明确专家能提供的价值

3. **描述组织/企业角色**
   - 确定实施环境的特征
   - 描述资源约束和目标
   - 明确决策标准

**每个角色定义应该包含：**
- 具体的背景信息
- 真实的需求和动机
- 相关的专业知识
- 独特的视角和价值
- 回应的风格特点

**请确保角色定义足够具体和真实，能够提供有价值的洞察。**
```

```markdown

### 实施维度
```

## 实施工具

### 1. 对话启动器

```
# 创意探索开始

你好！我有一个想法想和你探讨：[描述你的初始想法]

请帮我从不同角度深入思考这个想法：
1. 这个想法最有价值的地方是什么？
2. 谁会从中受益？
3. 需要什么资源才能实现？
4. 可能遇到什么挑战？

请随意提问，帮我深化和扩展这个概念。
```

### 2. 角色生成请求模板

```
# 角色生成请求

基于我们刚才的探索，请帮我生成三个AI角色的详细提示词：

1. **目标用户角色**：
   - 基于我们讨论的用户群体
   - 选择最典型或最有价值的用户类型
   - 让角色真实、具体、有代表性

2. **领域专家角色**：
   - 基于识别出的专业领域需求
   - 选择最关键的专家类型
   - 确保专家能提供专业洞察

3. **企业/组织角色**：
   - 基于实施环境的考虑
   - 反映真实的商业约束
   - 提供可行性评估

每个角色请使用前面提供的角色定义模板。
```

## 最佳实践

### 1. 对话技巧
- **保持开放**：不要过早下结论
- **追问细节**：对重要观点深入提问
- **记录洞察**：及时记录有价值的观点
- **连接想法**：寻找不同观点的联系

### 2. 角色设计
- **具体化**：避免泛泛而谈，给角色具体背景
- **真实性**：基于真实世界的人设计角色
- **多样性**：确保角色有不同视角
- **平衡性**：避免过于极端的观点

### 3. 验证方法
- **多轮对话**：通过多轮对话完善理解
- **场景测试**：在不同场景下测试想法
- **交叉验证**：让不同角色互相验证观点
- **真实反馈**：找真实的用户验证发现

## 常见陷阱

### 1. 探索阶段
- **范围过窄**：过早限制思考范围
- **确认偏误**：只寻找支持自己想法的证据
- **表面思考**：没有深入挖掘根本原因

### 2. 角色生成
- **刻板印象**：依赖对群体的刻板印象
- **过度简化**：忽略角色的复杂性
- **视角单一**：多个角色实际观点相同

### 3. 对话验证
- **引导性问题**：问题暗示特定答案
- **选择性倾听**：只听到想听的内容
- **缺乏批判**：不对观点进行质疑

## 进阶技巧

### 1. 角色演化
- 随着项目进展，更新角色定义
- 根据新信息调整角色特征
- 让角色"成长"和"学习"

### 2. 对话策略
- 使用假设性场景测试想法
- 让角色互相辩论和讨论
- 引入外部事件或约束

### 3. 系统整合
- 建立角色数据库重用成功角色
- 开发自动化角色生成工具
- 创建角色效果评估体系
